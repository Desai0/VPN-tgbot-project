# 🛡️ VPN Telegram Bot (Hysteria 2)

Автоматизированная система продажи и управления доступом к личному VPN-серверу через Telegram-бота. Проект построен на базе современного высокоскоростного протокола **Hysteria 2** и асинхронного фреймворка **aiogram 3.x**.

## 📌 Идея продукта
Цель проекта — создать удобный микросервис для предоставления услуг VPN с минимальным порогом входа для пользователя. 
Взаимодействие клиента с сервисом (оплата, получение настроек, контроль сроков подписки) происходит полностью через интерфейс Telegram-бота. Управление пользователями на сервере осуществляется динамически через REST API Hysteria 2 без необходимости перезагрузки VPN-ядра.

## 🚀 Основной функционал (MVP)
* Покупка и автоматическое продление VPN-подписки.
* Динамическое создание, изменение лимитов и удаление пользователей через API Hysteria 2.
* Автоматическая генерация конфигурационных ссылок (URI) для популярных клиентов (Nekobox, v2rayN и др.).
* Личный кабинет пользователя в боте (проверка статуса подписки, остатка дней).
* Полностью автоматизированное развертывание (CI/CD) через GitHub Actions.

## 🏗 Архитектура проекта (Монорепозиторий)
Проект разделен на логические компоненты:
* `/bot` — Пользовательский интерфейс: Telegram-бот на **aiogram 3**, обработка команд, меню, платежи.
* `/backend` — Бизнес-логика и API: взаимодействие с базой данных и REST API Hysteria 2.
* `/infra` — Инфраструктура: конфигурационные шаблоны (`.example`), `docker-compose.yml`, скрипты деплоя.
* `.github/workflows` — CI/CD пайплайны для автодеплоя на сервер.

## 🛠 Технический стек
* **Язык программирования:** Python 3.11+
* **Фреймворки:** aiogram 3.x, FastAPI / Aiohttp (для бэкенда)
* **VPN-ядро:** Hysteria 2
* **База данных:** PostgreSQL / SQLite + SQLAlchemy/asyncpg
* **Инфраструктура:** Docker, Docker Compose
* **CI/CD:** GitHub Actions
